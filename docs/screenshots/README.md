# Screenshots

Place the following screenshots in this folder:

| Filename | Description | Recommended Size |
|----------|-------------|------------------|
| `home-view.png` | Home screen with drive selection | 1920x1080 |
| `scanning-view.png` | Scanning progress display | 1920x1080 |
| `dashboard-treemap.png` | Dashboard with treemap visualization | 1920x1080 |
| `dashboard-tree.png` | Dashboard with directory tree view | 1920x1080 |
| `dashboard-table.png` | Dashboard with file table view | 1920x1080 |
| `extension-stats.png` | Extension analytics panel | 1920x1080 |
| `settings-view.png` | Settings configuration panel | 1920x1080 |

## Capture Guidelines

1. Use a clean installation with sample data
2. Ensure the window is maximized or at 1920x1080
3. For dark/light themes, capture both if desired
4. Use PNG format for lossless quality
5. Avoid capturing personal or sensitive file names
